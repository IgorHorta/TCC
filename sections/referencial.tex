\subsection{Javascript}
Javascpript é uma linguagem de programação interpretada com suporte a orientação a objetos, ela oferece recursos interativos que completam o HTML permitindo a criação de páginas iterativas e dinâmicas, que podem ser interpretadas localmente, sem necessitar de uma execução remota de programas no serviodor.
O javascript fornece ao autor a possibilidade de:

\begin{itemize}
\setlength{\itemsep}{-0.3ex}
\item Realizar validações de informações em formulários HTML.
\item Gerar documentos com aparências definidas a partir de informações vindas do backend ou do navegador do usuário.
\item Interagir com o conteúdo do documento, alterando propriedades da página, dos elementos HTML e tratando toda a página como uma estrutura de objetos.
\end{itemize}


Muitas vezes  javascprit  é confundido com a linguagem de programação java, apesar dos nomes semelhantes, as duas linguagens são extremamente diferentes, enquanto o javascript possui uma boa capacidade de controlar comportamentos do browser e conteúdo, o java trabalha bem no backend com networking e  multithreading.

\subsection{Java}
A linguagem de programação java  começou a ser desenvolvida em 1991 com a finalidade de ser utilizada em dispositivos eletrônicos inteligentes, porem com  o crescimento exponencial da World Wide Web em 1993, os engenheiros da Sun Microsystems notaram o potencial da linguagem para aprimorar a funcionalidade de servidores web e a adaptou para estas funções.
A seguir algumas vantagens da linguagem de programação java:

\begin{itemize}
\setlength{\itemsep}{-0.3ex}
\item Possui milhares de bibliotecas já implementadas para diversas funcionalidades, tais bibliotecas poupam significativamente o trabalho do desenvolvedor.
\item A tecnologia java oferece uma plataforma de desenvolvimento orientada a objetos limpa e eficiente.
\item É uma linguagem simples, fácil de aprender e que não exige um conhecimento muito grande do desenvolvedor para se ter produtividade.
\end{itemize}

\subsection{MySQL}
SGBD ou sistema gerenciador de banco de dados, é um conjunto de programas que tem como objetivo armazenar,alterar e apagar informações de uma base de dados. Estes são bastante uteis em aplicações web, por permitirem que as informações sejam armazenadas e utilizadas automaticamente.
O MySQL é um popular SGBD openSource, que se destaca perante as outras opções no mercado por possuir um excelente desempenho.
O MySQL também possui as seguintes vantagens:

\begin{itemize}
\setlength{\itemsep}{-0.3ex}
\item Portabilidade.
\item Utiliza a linguagem SQL.
\item Compatibilidade com diversas linguagens de programação.
\item Estável e com excelente desempenho.
\item Pouco exigente quanto aos recursos de  hardware.
\item Fácil instalação e configuração
\end{itemize}
