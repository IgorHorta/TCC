\subsection{Contextualização e Motivação}

Com o advento da Internet, surgiu também uma nova espécie de comércio, o chamado \emph{e-commerce} ou comércio digital, que possui a vantagem de não limitar geograficamente o consumidor e disponibilizar para o mesmo um maior conjunto de lojas e produtos.
De acordo com a E-bit, as vendas no comércio eletrônico em 2014 no Brasil continuaram crescendo e atingiram um resultado além do esperado. O faturamento do setor \emph{e-commerce} arrecadou com vendas o equivalente a R\$35,8 bilhões. O que representa um crescimento de 24\% em relação a 2013, quando se vendeu um total de R\$28,8 bilhões \cite{EBIT2015}. 


O número de pedidos também aumentou, segundo a \cite{EBIT2015}, o aumento deste em 2014 foi de 17\% em relação ao ano
anterior, chegando a 103,4 milhões. Em 2013 foram 88,3 milhões de encomendas de bens de consumo via Internet. Em 2015, espera-se que o número de encomendas seja 19\% maior, chegando a 122,9 milhões.

Com o aumento dos requisitos mínimos para se utilizar jogos e programas atuais e com o aumento de possíveis combinações de peças de computadores, a tarefa de customizar seu próprio computador para executar funções especificas se torna cada vez mais difícil.

Tendo como base o crescente sucesso e procura do comércio eletrônico e consequentemente o aumento de \emph{websites} de \emph{e-coomerce}, este trabalho propõe um sistema para facilitar a tarefa de customizar seu próprio computador. Tal sistema utilizará técnicas de extração de dados para obter preços e informações sobre diversas peças de computadores de diferentes sites especializados em comércio eletrônico. Com tais informações disponíveis o sistema proposto irá disponibilizar diversos recursos como comparação de preços entre peças e auxílio na customização de computadores visando o melhor custo benefício. 



\subsection{Objetivos Gerais e Específicos}
O objetivo geral deste trabalho é construir uma aplicação web que auxilie o consumidor a comprar e montar seu computador personalizado. Esta aplicação possuirá o recursos de comparação de preços de peças de computadores das principais lojas de \emph{e-commerce} desta categoria.

Visando atingir este objetivo geral, os objetivos específicos são definido a seguir:
\begin{itemize}
\setlength{\itemsep}{-0.3ex}
\item Desenvolver um algoritmo eficiente que seja capaz de coletar informações de produtos em diferentes sites de \emph{e-commerce}.
\item Extrair as informações sobre produtos de pelo menos 5 sites que vendem peças de computadores.
\item Modelar e criar o banco de dados da aplicação, integrando as informações coletadas dos diferentes sites com os dados internas da aplicação desenvolvida.
\item Desenvolver um protótipo de interface para aplicação (\emph{website}).
\item Implementar e tornar disponível para os consumidores um sistema capaz de montar um computador customizado e buscar os melhores preços para todas as peças.
\end{itemize}



\subsection{Metodologia}
Será desenvolvida uma aplicação para realizar comparações de preços entre peças de computadores de diversas lojas de \emph{e-commerce},
tal aplicação aprestara recursos para auxiliar o usuário a montar seu próprio \emph{desktop} customizado com o melhor custo benefício.


Um algoritmo para extração de dados será utilizado para se obter informações como: preço, imagem e descrição de produtos, tal algoritmo será escrito em Javascript e será executado utilizando-se o PhatomJS. O \emph{scprit} obterá a árvore DOM do \emph{website} e ai então buscará por padrões que possam identificar uma lista de produtos.
Se tal lista for encontrada com sucesso a mesma sera enviada para a aplicação, que será responsável por verificar a integridade dos dados e então salva-la no banco de dados.

Depois de obter os dados o próximo passo será criar uma interface amigável, tal  interface será criada utilizando-se HTML,CSS e Javascript e possuirá recursos para listar os produtos obtidos com seus respectivos preços.
